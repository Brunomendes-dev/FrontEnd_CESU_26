Fundamentos e Arquitetura de Computadores

1. A Unidade Central de Processamento (CPU)

A CPU é o componente que executa as instruções de um programa. Ela é dividida basicamente em três partes:

  • UC (Unidade de Controle): Coordena a movimentação de dados e instruções.
  • ULA (Unidade Lógica e Aritmética): Onde os cálculos matemáticos e testes lógicos acontecem.
  • Registradores: Pequenos espaços de memória ultra-rápidos dentro do processador.

2. Modelo de Von Neumann

É a base da maioria dos computadores. O conceito principal é que tanto as instruções (o software) quanto os dados são armazenados na mesma memória principal.

O Ciclo de Instrução:
1. Busca (Fetch) na memória -> 2. Decodificação (Decode) -> 3. Execução (Execute).

3. Hierarquia de Memória

Como não existe uma memória que seja rápida, barata e enorme ao mesmo tempo, a arquitetura utiliza uma hierarquia:

  • Nível 1: Registradores (Mais rápidos / Menor capacidade)
  • Nível 2: Memória Cache (L1, L2, L3)
  • Nível 3: Memória Principal (RAM)
  • Nível 4: Armazenamento Secundário (SSD, HD)

4. Barramentos (Bus)

São as vias de comunicação que interligam os componentes. Existem três tipos principais:

  • Barramento de Dados: Transporta a informação em si.
  • Barramento de Endereço: Indica para onde a informação deve ir ou de onde deve vir.
  • Barramento de Controle: Sinais que coordenam as operações (leitura, escrita, etc).
